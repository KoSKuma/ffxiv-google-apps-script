# Release v1.0.0 - Initial Release


**Release Date:** 2025-12-28

## üéâ First Stable Release

This is the first stable release of the FFXIV Google Apps Script project, providing powerful automation tools for managing Final Fantasy XIV crafting and material information directly in Google Sheets.

## ‚ú® Features

### ‚úÖ Item Information Lookup

Retrieve comprehensive item information including:
- **Gathering Locations**: Node names, levels, zones, and gathering types
- **Vendor Information**: NPC locations, prices (Gil and special currencies), and shop details
- **Aetherial Reduction Sources**: Items that can be reduced to obtain the target item
- **Batch Processing**: Process multiple items at once from spreadsheet columns

**Functions:**
- `lookupItemInfo(itemName)` - Single item lookup
- `processItemList(sheetName, itemColumn, startRow)` - Batch processing from spreadsheet

### ‚úÖ Crafting Request Processing

Batch process crafting requests with automatic material calculation:
- **Recursive Material Calculation**: Automatically includes sub-ingredients (ingredients that need to be crafted)
- **Quantity Multiplication**: Automatically multiplies material quantities by requested item quantity
- **Dynamic Row Insertion**: Automatically inserts rows for materials with proper formatting
- **Cycle Detection**: Prevents infinite loops in crafting recipes
- **Depth Limiting**: Limits recursion to 5 levels to prevent excessive API calls

**Function:**
- `processCraftingRequest(sheetName, itemColumn, quantityColumn, startRow)` - Batch process crafting requests

### üß™ Debug/Utility Tools

Additional tools for testing and debugging:
- `getCraftingMaterials(itemName, includeSubIngredients)` - Get crafting materials for a single item
- Individual item lookup and batch processing with manual parameters

## üöÄ Quick Start

```bash
npm install -g @google/clasp
clasp login
clasp create --type sheets --title "FFXIV Google Apps Script"
clasp push
```

See [GETTING_STARTED.md](../GETTING_STARTED.md) for detailed setup instructions.

## üìñ Usage

### As a Bound Spreadsheet

After pushing the code, open your Google Sheet and you'll see a custom menu "FFXIV Tools" with:
- **Process Crafting Request** - Batch process crafting requests from "Requested for Crafting" sheet
- **Obtain Material Information** - Process items from "Material Info" sheet
- **Debug** submenu - Testing and debugging tools

### As a Library

Deploy as a library to use across multiple spreadsheets:

1. In Apps Script editor: **Deploy** ‚Üí **New deployment** ‚Üí Select **Library**
2. Share Script ID with users
3. Users add library via: **Extensions** ‚Üí **Apps Script** ‚Üí **Libraries**
4. Copy menu code from `LIBRARY_TEMPLATE.gs` to their spreadsheet

See [GETTING_STARTED.md](../GETTING_STARTED.md) for detailed library setup.
